Инструкция к игре "Поле Чудес"

Запуск игры

Для запуска файла необходимо установить .NET Framework. Это можно сделать по ссылке: Скачать .NET Framework.

Начало игры

После открытия файла вы увидите следующее окно:

Для начала первого раунда игры нажмите на кнопку: “Начать игру”.

Каждый игрок по очереди крутит барабан, который может выпасть на разные сектора:

  • Обычные очки
  • Сектор «+» – открыть символ и получить 100 очков.
  • Сектор «Переход хода» – пропустить ход.

Игровой процесс

В строке «Слово: » загадано слово, которое игроки должны отгадать. Подсказка к загаданному слову находится в верхней части программы.

Слева находится таблица с балансом всех игроков. После успешного угадывания символа, очки зачисляются игроку.

Результаты

Последний игрок, который ввел правильный символ, побеждает. Выводится окно с номером победившего игрока и количеством его очков.

После нажатия на кнопку «ОК» предлагается начать новую игру.
